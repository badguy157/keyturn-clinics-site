## ðŸŽ¨ General Prompt Structure Tips

### Best Practices for AI Image Generation:
1. **Be Specific:** Include details about lighting, colors, perspective, style
2. **Set Aspect Ratio:** Use `--ar 16:9` for landscape, `--ar 4:5` for portrait
3. **Specify Quality:** Add terms like "8k", "ultra realistic", "professional photography"
4. **Include Style References:** "architectural digest style", "commercial photography", "editorial"
5. **Color Palette:** Mention your brand colors (navy blue, gold accents)
6. **Avoid Common Issues:** Specify "no text", "no watermarks" if needed

### Platform-Specific Parameters:
- **Midjourney:** Use `--ar 16:9 --v 6 --style raw` for realistic photos
- **DALL-E 3:** Natural language works best, be conversational
- **Stable Diffusion:** Use model-specific keywords, separate with commas
- **Leonardo AI:** Mention "photorealistic" and use Alchemy for quality

## ðŸ”„ Iteration Tips

### If First Generation Isn't Perfect:
1. **Too Dark/Bright:** Add "soft even lighting" or "professional studio lighting"
2. **Wrong Angle:** Specify "shot from slightly above" or "eye-level perspective"
3. **Too Busy:** Add "minimalist", "clean", "negative space"
4. **Not Realistic:** Add "photorealistic", "ultra realistic", "real photography"
5. **Wrong Colors:** Be more specific: "deep navy blue (#0D1B2A)", "metallic gold (#D4AF37)"

### Remix Commands (Midjourney):
- Add `--chaos 20` for more variety
- Use `--stylize 100` for more creative interpretation
- Try `--weird 50` for unique perspectives
