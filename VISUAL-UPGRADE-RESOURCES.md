Lottie files are JSON-based animations exported from Adobe After Effects. They're scalable, lightweight, and performant.

**Use Case:** Service icons, feature highlights

### Lottie Implementation:

#### Method 1: Lottie Player (Easiest)
```html
<!-- Add Lottie Player Script -->
<script src="https://unpkg.com/@lottiefiles/lottie-player@latest/dist/lottie-player.js"></script>

<!-- Embed Animation -->
<lottie-player
  src="path/to/animation.json"
  background="transparent"
  speed="1"
  style="width: 300px; height: 300px;"
  loop
  autoplay>
</lottie-player>
```

#### Method 2: Lottie Web Library
```html
<!-- Add Library -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/lottie-web/5.12.2/lottie.min.js"></script>

<!-- Container -->
<div id="lottie-animation"></div>

<!-- Initialize -->
<script>
  lottie.loadAnimation({
    container: document.getElementById('lottie-animation'),
    renderer: 'svg',
    loop: true,
    autoplay: true,
    path: '/animations/process-flow.json'
  });
</script>
```

### Download & Integration Steps:
1. Visit LottieFiles.com
2. Search for desired animation
3. Click "Download" ‚Üí Select "Lottie JSON"
4. Save to `/animations/` folder
5. Implement using Lottie Player or Web library
